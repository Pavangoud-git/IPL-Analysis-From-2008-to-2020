# IPL Analysis Dashboard
Overview

This project presents an interactive dashboard for analyzing data from the Indian Premier League (IPL). The dashboard helps users explore match statistics, team performance, and player achievements across different seasons. It provides clear visual insights into how factors such as toss decisions and player performance influence match outcomes.

The goal of this project is to demonstrate data analysis and visualization skills by transforming raw IPL data into meaningful insights using an interactive dashboard.

# Features

Season-wise analysis of IPL matches

Toss decision impact on match results

Visualization of total fours and sixes scored in the tournament

Identification of Orange Cap (highest run scorer) and Purple Cap (highest wicket taker) winners

Interactive filters to explore different seasons

Clear visual representation of match outcomes and team performance

# Tools & Technologies Used
